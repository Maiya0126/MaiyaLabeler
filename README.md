介绍

你是否厌倦了地图上全是枯燥的“卧室”、“仓库”？
麦丫标签机 让你能够直接在游戏地图上，为每一个房间、种植区和储存区添加可视化的自定义标签。

主要功能
可视化标签：直接在地图表面显示房间和区域的名字，字体清晰，位置自适应。
高度自定义：
- 改名：把“餐厅”改成“米其林三星”，把“医疗室”改成“协和医院”。
- 改色：内置16种常用色，直接选择。
- 改样式：调整透明度、字号大小、位置偏移。

- 新增添加备注功能（类似描述）：房间、区域添加备注后，可以让RimTalk读取到。（例如“这是一间充满了科技感的餐厅，墙上挂着量子相框，空气中弥漫着草莓味营养膏的香气”，RimTalk 的 AI 就能根据这些细节生成无比生动的对话。）

图标系统：为不同类型的区域显示原版图标，支持一键开关。
RimTalk联动：
- 如果你安装了 RimTalk 模组，当你把餐厅改名为“米其林三星”后，小人们在聊天时会真的说出这个名字！
- （例如：“我们要不要去米其林三星餐厅搓一顿？”）
安全稳定：
- 针对大型整合包优化，包含“3秒安全启动锁”，杜绝进地图崩溃。
- 对性能几乎无影响。
- 可以随时中途加入或删除。

如何使用
1. 开关显示：点击右下角的一排按钮中的“麦丫标签机（重命名图标）”，或者快捷键，或者右键点击底部标签机按钮。
2. 配置标签：在【区域】菜单中选择“配置标签”工具，或者快捷键，或者左键点击底部标签机按钮，然后点击任意房间或区域使用。

兼容性
- 支持 RimWorld 1.5、1.6 版本。
- 完美兼容 RimTalk（软兼容，不强制安装）。
- 界面缩放已处理。

语言
- 支持中文、英文。
- 考虑日韩俄三语。
